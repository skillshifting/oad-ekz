# 25. Хранение данных в БД. СУБД, OLAP сценарий

### **СУБД (система управления базами данных) — это программное обеспечение, с помощью которого можно создавать, хранить, изменять и анализировать данные**.

### Проще говоря, СУБД — **промежуточное звено между самими данными и теми, кто с ними работает** — разработчиками, аналитиками или приложениями.


---

## БД (Базы данных):

* Реляционные
* Строковые
* Столбчатые

---

## OLAP сценарий


### **OLAP-сценарий** — это способ работы с большими данными, который позволяет легко анализировать информацию с разных сторон.

Огромный куб данных (например, продажи товаров в разных магазинах за год). 

С помощью OLAP можно:

* смотреть на данные по месяцам, кварталам или годам;
* анализировать продажи по разным магазинам;
* изучать популярность отдельных товаров;
* комбинировать эти параметры как угодно.

Из особенностей, что данные обновляются большими пачками, большинство запросов на чтение, в запросе одна большая таблица, результат запроса меньше исходных данных.
