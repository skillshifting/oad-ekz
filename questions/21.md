# 21. Типовые проблемы в источниках данных


1. Способ хранения данных
2. Ручной ввод данных в некоторых ячейках
3. Объединённые ячейки
4. Неоднородная структура хранения данных
5. "Смешать, но не взбалтывать" - листы таблицы могут быть смешаны, например: (январь/февраль/апрель/.../1 полугодие/2 полугодие)
6. Суммарные значения некоторых ячеек
7. Разные показатели ячеек, которые не сходятся в конце концов
